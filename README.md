# NTKMA Media Intelligence Dashboard

NTKMA Media Intelligence is an institutional memory and governance tracking platform for the Nashik–Trimbakeshwar Kumbh ecosystem.

The platform helps monitor:
- Print media coverage
- Governance narratives
- Stakeholder mentions
- Sentiment trends
- Repetitive issues
- Risk signals
- Infrastructure and policy discussions

## Features

- Daily media entry system
- Sentiment classification
- Theme classification
- Repetition tracking
- Stakeholder mapping
- Weekly summary generation
- CSV export
- JSON backup system
- Keyword-based news monitoring

## Use Case

The dashboard is designed for:
- Researchers
- Urban planners
- Journalists
- Policy analysts
- Kumbh governance observers
- Administrative monitoring teams

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser local storage
